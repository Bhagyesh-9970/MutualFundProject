# Project Structure

## Root folder

- app.py: Streamlit dashboard entry point
- efficient_frontier.py: portfolio optimization logic
- monte_carlo.py: NAV simulation engine
- email_report.py: weekly report generator
- requirements.txt: Python dependencies
- schedule_nav_etl.ps1: Windows scheduling script

## Data folders

- data/raw: original source CSV files
- data/processed: cleaned and ready-to-use datasets

## Analysis folders

- notebooks: exploratory analysis notebooks
- scripts: ETL, cleaning, validation, and database utilities
- sql: schema and query files
- tests: regression tests for the bonus modules

## Reporting folders

- reports: generated summaries, outputs, and charts
- dashboard: Power BI instructions and theme assets
- docs: project documentation
