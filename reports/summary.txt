# Mutual Fund Analytics Platform

### Bluestock Fintech – Individual Capstone Project

---

# Project Description

The **Mutual Fund Analytics Platform** is a full-stack data engineering and analytics solution designed to consolidate fragmented Indian mutual fund data into a single analytical platform. The project integrates data from public sources such as **AMFI India** and **mfapi.in**, processes it through an ETL pipeline, stores it in a relational database, performs financial and risk analysis, and presents insights using interactive dashboards.

The platform enables investors and analysts to track NAV movements, analyze AUM trends, study investor behavior, compute risk-adjusted returns, and compare mutual funds with benchmark indices.

---

# Aim

To build an end-to-end Mutual Fund Analytics Platform that automates data ingestion, cleaning, storage, analysis, and visualization of Indian mutual fund data for informed investment decision-making.

---

# Objectives

### O1. Build an ETL Pipeline

* Extract mutual fund datasets and live NAV data from public APIs.
* Clean, transform, and validate the data.
* Automate data processing using Python.

### O2. Design a Relational Database

* Create a star-schema database model.
* Store data efficiently using SQLite/PostgreSQL.
* Maintain relationships between dimension and fact tables.

### O3. Perform Exploratory Data Analysis

* Analyze NAV trends and AUM growth.
* Study category-wise and fund-wise performance.
* Generate statistical summaries and visualizations.

### O4. Compute Financial Risk Metrics

* Calculate Sharpe Ratio, Sortino Ratio, Alpha, Beta.
* Measure Max Drawdown and Value-at-Risk (VaR).
* Compare funds with benchmark indices.

### O5. Analyze Investor Behaviour

* Study SIP, lumpsum, and redemption transactions.
* Perform demographic and geographic analysis.
* Identify investment patterns across states and age groups.

### O6. Build Interactive Dashboards

* Develop Power BI/Tableau dashboards.
* Create KPI cards and drill-down reports.
* Provide fund comparison and performance views.

### O7. Benchmark Mutual Fund Performance

* Compare schemes with Nifty 50, Nifty 100, and BSE SmallCap.
* Compute tracking error and alpha.
* Evaluate fund outperformance.

### O8. Deliver Complete Documentation

* Maintain GitHub repository.
* Create data dictionary and SQL scripts.
* Prepare reports and presentation decks.

---

# Problem Statement

Indian mutual fund data is scattered across multiple sources and formats, making investment analysis difficult. Investors and analysts lack a unified platform to evaluate fund performance, compare schemes, and understand investor behavior. This project addresses these challenges by creating a centralized analytics platform with automated ETL, database storage, risk analytics, and business intelligence dashboards.
